ST(Sparse table)表

一维：

以求最大值为例，设 $d[i,j]$ 表示 $[i,i+2^{j-1}]$ 这个区间内的最大值，那么在询问$[a,b]$区间的最大值时答案就是$max(d[a,k], \ \ d[b-2^k+1])$，其中 $k$ 是满足 $2^k<=b-a+1$ (即长度)的最大的 $k$ ,即 $k=[ln(b-a+1)/ln(2)]$ 。

Description: 在给定的 $N$ 个整数 $A_1，A_2\dots A_N$ 中选出两个进行 XOR（异或）运算，得到的结果最大是多少？

Solution：一个整数，是可以转化成为一个32位的二进制数。故查询与 $a[i]$ 异或值最大的数时，我们都走与当前这一位相反的位置，也就是让Xor值最大，如果说没有路可以走的话，那么就走相同的路.

主席树

查询区间有多少个不同数

给定长度为 $N$ 的整数序列 $A$，下标为 $1∼N$。

现在要执行 $M$ 次操作，其中第 $i$ 次操作为给出两个整数 $l_i,r_i$ 求 $A[l_i],A[l_i+1],…,A[r_i]$(即 A 的下标区间 $[l_i,r_i])$中有多少个不同的数。

Solution：

对于一个询问的区间[l,r]，如果区间中出现的多个同一数字，一定是只关心出现在最右边的那一个的数。用 $cnt[i]$ 表示序列的第 $i$ 个数在当前询问中是否为最靠右的那一个数（可以当作答案统计的那一个数）。建立 n 颗线段树，每次查找答案在对应标号的线段树上查找即可。

区间第k小问题

给定长度为 $N$ 的整数序列 $A$，下标为 $1∼N$。

现在要执行 $M$ 次操作，其中第 $i$ 次操作为给出三个整数 $l_i,r_i,k_i$ 求 $A[l_i],A[l_i+1],…,A[r_i]$(即 A 的下标区间 $[l_i,r_i])$中第 $k_i$小的数是多少。

Solution:

发现以$[1,x]$和$[1,x+1]$区间内的数所建立的权值线段树的差异仅在一条链上：（$x+1$的次数）。

也就是不超过$log_2n$个点。我们可以考虑动态开点：

  • 与上一个权值线段树没有差异的地方直接指引过去

  • 有差异，单独新增一个点

这样即可预处理出$[1,x]$所有的权值线段树了。

时间复杂度$O(nlog_2n)$，空间复杂度$O(2n+nlog_2n)$。

平衡树

需要写一种数据结构，来维护一些数，其中需要提供以下操作：

  1. 插入数值 x。
  2. 删除数值 x(若有多个相同的数，应只删除一个)。
  3. 查询数值 x 的排名(若有多个相同的数，应输出最小的排名)。
  4. 查询排名为 x 的数值。
  5. 求数值 x 的前驱(前驱定义为小于 x 的最大的数)。
  6. 求数值 x 的后继(后继定义为大于 x 的最小的数)。
